# DMED — Медицинская информационная система

Next.js 14 приложение (React 18, TypeScript).

## Требования

- **Node.js** 18.17 или новее
- **npm** 9+

## Запуск

1. **Установка зависимостей**

   В корне проекта выполните:

   ```bash
   npm install
   ```

2. **Режим разработки**

   ```bash
   npm run dev
   ```

   Приложение будет доступно по адресу: [http://localhost:3000](http://localhost:3000)

3. **Сборка для продакшена**

   ```bash
   npm run build
   npm start
   ```

## Скрипты

| Команда       | Описание                    |
|---------------|-----------------------------|
| `npm run dev` | Запуск dev-сервера          |
| `npm run build` | Сборка для продакшена     |
| `npm start`   | Запуск после `npm run build`|
| `npm run lint`| Проверка кода (ESLint)      |

## Окружение

Переменные окружения (при необходимости) задаются в файле `.env` в корне проекта. Локальные переопределения — в `.env.local`.

## Структура

- `src/app/` — страницы и маршруты (App Router)
- `src/features/` — фичи (auth, dashboard и др.)
- `src/shared/` — общие сервисы и хуки
- `src/widgets/` — виджеты и разметка
- `src/types/` — типы TypeScript
